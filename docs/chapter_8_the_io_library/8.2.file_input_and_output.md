*fstream* header 提供了3个类型来支持文件IO:  
- ifstream读文件
- ofstream写文件  
- fstream读写

#### 8.2.1. Using File Stream Objects

```cpp
ifstream ifs(filepath);
```
读取文件到ifstream之后, 可以将其转换为string.  
每个单词执行: `ifs >> word`  
每行执行: `getline(ifs, line)`  
整个文件一次性执行呢? 如下:
```cpp
ostringstream oss;
oss << ifs.rdbuf();
string s = oss.str();
```
